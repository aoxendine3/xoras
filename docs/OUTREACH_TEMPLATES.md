# XORAS: Outreach Templates

Use these templates for initial pilot outreach to engineering leaders, founders, and CTOs.

---

## 1. The Frictionless Founder DM (Slack / LinkedIn / Discord)
**Use Case:** Direct outreach to 5-30 person engineering teams.

> Would your team be interested in a 30-day XORAS Release Integrity pilot?
>
> XORAS runs in Advisory Mode during the pilot — no build blocking or workflow disruption. It surfaces release drift signals like latency regressions, dependency bloat, and exposed secrets directly inside CI so teams can improve release confidence before enforcement is ever introduced.
>
> Setup takes only a few minutes through a GitHub Action. The pilot includes:
>
> * Prevented incident tracking
> * Weekly governance summaries
> * Release drift analytics
> * Direct onboarding support
>
> Happy to share the demo repo and walkthrough if you'd like to see it in action.
